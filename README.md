# Smart Study Planner

## Description
A Python-based application that generates a smart study timetable based on subject priority and study hours.

## Features
- Input subjects and study hours
- Assign priority levels
- Automatically sorts subjects
- Generates timetable with time slots (Morning, Afternoon, Evening)

## Technologies Used
- Python

## How to Run
1. Install Python
2. Run: python study_planner.py
